# 🧠 Simulacro Backend

Backend de una aplicación de simulacros de examen. Desarrollado con Node.js, Express, TypeScript y Prisma ORM con base de datos SQLite.

## 🚀 Funcionalidades

- Registro e inicio de sesión con JWT
- Simulacros con duración fija (120 minutos)
- Banco de preguntas (opción múltiple)
- Respuesta y finalización de simulacros
- Recompensas (experiencia y monedas)
- Gestión de perfil de usuario
- Cambio de contraseña
- Registro y listado de preguntas (modo administrador)

---

## 🛠️ Tecnologías

- [Node.js](https://nodejs.org/)
- [Express](https://expressjs.com/)
- [TypeScript](https://www.typescriptlang.org/)
- [Prisma ORM](https://www.prisma.io/)
- [SQLite](https://www.sqlite.org/)
- [JWT (jsonwebtoken)](https://github.com/auth0/node-jsonwebtoken)
- [bcryptjs](https://github.com/dcodeIO/bcrypt.js)
